龙年第一天，历时逾半年的龙架构移植已完成合并，正式“合龙”！

该架构软件源已从 `frontier` 移至 `stable`，后续更新将推送到此仓库。

目前已经安装龙架构系统的用户，请运行如下命令切换软件源：

```bash
oma mirror reset
```

如需切换镜像源，请使用如下命令：


```bash
oma mirror
```

在未来几天，我们将对龙架构移植进行最后修缮，预计在年假内将发布龙架构的第一批系统发行，敬请期待！
